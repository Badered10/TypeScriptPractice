# TypeScript Learning Repository

This repository is dedicated to learning and experimenting with **TypeScript**, a typed superset of JavaScript that helps in writing safer and more maintainable code.

## About
The goal of this repository is to explore and practice various TypeScript concepts, including:

- Basic types and interfaces
- Functions and generics
- Classes and inheritance
- Modules and namespaces
- Working with third-party libraries

Each folder or file may focus on a specific concept, exercise, or small project.
